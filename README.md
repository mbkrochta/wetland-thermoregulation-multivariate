# wetland-thermoregulation-multivariate
Scripts and data for Krochta M., Chang H., & Zaret K. (in review). Characterizing wetland thermoregulation using multivariate water temperature patterns and site-level functional indicators. Journal of Freshwater Ecology.
All analyses were conducted in R using the scripts provided within subdirectories.
Raw temperature data are not included, but all processed data needed to reproduce the analyses are provided.
Please note that this repository accompanies a manuscript currently under peer review and may be updated following revision or acceptance.
